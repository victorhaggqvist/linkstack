get /items

Get a list of items

  • Request
  • Response

Query Parameters

  • query: (string)

    A search query

    Example:

    my query string
  • Page: (integer)

    The page to get

  • Items Per Page: (integer)

    Number of items to receive.

  • Sort: one of (ASC,DESC)

    Specifies the sort order of the list.

  • Key: required (string)

    Your api key

  • tags: one of (1,0)

    Query only tags
